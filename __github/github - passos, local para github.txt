4. workflows (local)

. cria uma pasta dentro do projeto audio-player-playlist-react:
. pasta: .github
. nova pasta: workflows dentro da pasta .github
.github > workflows (cria arquivo build.yml) conteudo abaixo:

name: deploy

on:
  push:
    branches:
      - master

jobs:
  deploy:
    runs-on: ubuntu-18.04
    steps:
      - uses: actions/checkout@v2
      - uses: actions/setup-node@v1
        with:
          node-version: '14.x'
      - name: Build web-app
        run: |
          npm ci
          npm run build
      - name: Deploy to gh-pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./build

5. git add ./src
6. git commit -m 'add task solution'
7. git push origin develop (ou master)
8. npm run deploy
